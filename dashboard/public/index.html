System Overview

The tv-oracle-bridge acts as an institutional-grade bridge connecting TradingView indicators, data feeds, and local backtesters.

  • Delta Syncing SQLite Cache: Loads historical periods, prevents duplicate calls, and updates on new closes.
  • Canvas Annotation Overlays: Automatically highlights candlestick patterns directly on PNG charts.
  • Pine Script Linter & Docs: Full dictionary covering v5/v6 commands with spellcheck suggestion algorithm.
  • Automated Script Downloader: Pulls source code from any public TradingView community script.
