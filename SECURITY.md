# Security Policy

## Reporting a Vulnerability

We take the security of this project seriously. If you discover a security vulnerability in this project, please report it privately. **Do not open a public issue.**

To report a vulnerability, please contact the maintainer directly through the contact details on their GitHub profile ([@andreafinazziinfo](https://github.com/andreafinazziinfo)).

Please include the following information in your report:
- A detailed description of the vulnerability.
- Step-by-step instructions to reproduce the issue.
- Potential impact or exploit scenarios.

We will acknowledge receipt of your report within 48 hours, investigate the issue, and coordinate a fix before making a public disclosure.
